## What is TUNA?

TUNA is a decentralized news aggregator designed specifically for the Sui ecosystem. It brings together news from various sources and stores them on the Sui blockchain and Walrus decentralized storage network, enabling users to engage with content through tipping and commenting.

## How It Works

TUNA uses a hybrid architecture combining the Sui blockchain for indexing and transactions with Walrus for content storage:

1. **News Registry** - A smart contract on Sui maintains an index of all news articles via blob IDs
2. **Walrus Storage** - Full article content is stored on the Walrus decentralized storage network
3. **Engagement System** - Tips and comments are tracked on-chain with engagement metrics
4. **React Frontend** - A modern React application provides the user interface

<Note>
TUNA is currently deployed on the Sui testnet. Make sure you have testnet SUI tokens to interact with the application.
</Note>

## Why TUNA?

Traditional news aggregators are centralized and controlled by a single entity. TUNA brings several advantages:

- **Decentralization** - No single point of control or censorship
- **Incentive Alignment** - Tip quality content to incentivize good journalism
- **Permanent Storage** - Articles stored on Walrus are immutable and permanent
- **Transparent Engagement** - All tips and interactions are recorded on-chain
- **Open Integration** - Anyone can build on top of the TUNA protocol

## Tech Stack

TUNA is built with modern web3 technologies:

- **Frontend**: React 19, TypeScript, Vite
- **Blockchain**: Sui Network (Testnet)
- **Storage**: Walrus Decentralized Storage
- **State Management**: TanStack Query (React Query)
- **Wallet**: @mysten/dapp-kit
- **Blockchain SDK**: @mysten/sui.js
